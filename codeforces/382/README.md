## C
**OGP,斐波那契,递推**

你注意到

- 打1场比赛,需要2个人
- 打2场比赛,需要打过1场比赛的人和一个打过0场比赛的人,要3个人
- 打3场比赛,需要打过2场比赛的人和一个打过1场比赛的人,需要5个人
- 打4场比赛,需要打过3场比赛的人和一个打过2场比赛的人,要8个人

依次类推,得到序列2,3,5,8,13,21,...

给定n个人,如果最快打完x场比赛需要n<sub>x</sub>个人,</br>
如果n<sub>x</sub> &le; n &lt; n<sub>x+1</sub>,</br>
那么输出x
