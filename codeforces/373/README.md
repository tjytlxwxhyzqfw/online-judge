## C
**浮点字符串**

4舍5入,好的模拟题.

你注意到:

1.应该始终从小数点右边第一个可以进位的位置进位

2.除非x本来就是9,否则再任何情况下,x都绝不会向它的左边进位

###### 小技巧
- 独立的四舍五入函数
- 独立的进位函数
- 去掉小数点,将字符串拼接在一起, 两个指针s,t分别指向整数和小数的起始部分
- 独立的打印函数

###### 打印浮点字符串
打印函数应该:

- 从最左端起,滤掉所有的前缀0
- 从最右端起,**向左**滤调所有的后缀0

然后应用以下两个原则打印字符串:

- 前缀小数点前加0,后缀小数点省略
- 空串为0
