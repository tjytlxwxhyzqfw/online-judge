###### 使用计数器

这题是可以这样做到遍历的:

1. 设D和R的计数器分别为x, y
2. 开始遍历序列
3. 若当前元素的计数器是0, 那么就增加另一个元素的计数器
4. 若当前元素的计数器是个正直, 那么就删掉当前元素
